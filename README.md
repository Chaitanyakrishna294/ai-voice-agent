# AI Voice Agent

## Overview

An AI-powered voice agent that converts speech to text and enables intelligent voice interactions using modern AI technologies.

## Tech Stack

* Frontend: Next.js, TypeScript
* Backend: Python
* Speech Recognition
* AI/LLM Integration
